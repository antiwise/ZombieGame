=General Description
A survival horror game fashioned like the typical 2D shooter. The object is to protect your lover Pink Tofu while killing as many as green Tofu (zombies!) as possible.

=Features
Offers multiple weapons and lots of explosions. In the sample SWF file, the weapons are unlocked and can be simply accessed by numpads 1-7. A* algorithms for the green tofu and a demonstration app showing A* calculation is available in a separate SWF.

=Notes
Features bare minimum amount of artwork. Everything is created and hand-drawn by me. A project completed at previous co-op.
